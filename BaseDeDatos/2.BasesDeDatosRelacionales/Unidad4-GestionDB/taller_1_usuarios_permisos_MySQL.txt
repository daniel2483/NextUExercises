Taller: Definiendo Usuarios y Permisos para empresa en MySQL

MySQL tiene ciertos comandos y sentencias para definir usuarios y sus correspondientes permisos, los cuales hemos estudiado anteriormente. El objetivo de este taller es que logres definir usuarios y permisos para la tabla productos de una empresa en MySQL. La tabla productos se encuentra a continuación:

Productos:
id (clave primaria) 	INT AUTOINCREMENTAL
nombre 	VARCHAR(256) NOT NULL
código 	INT
precio 	FLOAT

Realiza los siguientes pasos:

    Crea la tabla productos.
    Crea un usuario llamado ‘cliente’ con clave ‘miclavecliente’.
    Crea permisos para ‘cliente’ de solo SELECT para la tabla ‘productos’.
    Crea un usuario llamado ‘gerente’ con clave ‘miclavegerente’.
    Crea permisos para ‘gerente’ de todos los privilegios para la tabla ‘productos’.


Puntos de verificación

Para confirmar que tu experiencia ha sido llevada a cabo con éxito, revisa los siguientes puntos:

    Verifica la tabla: Revisa que tenga todos los campos.
    Conectáte con el usuario ‘cliente’.
    Realiza un SELECT y verifica que funciona.
    Realiza un INSERT y verifica que no está permitido.
    Vuelve a conectarte pero ahora con el usuario ‘gerente’.
    Realiza un SELECT y verifica que funciona.
    Realiza un INSERT y verifica que también funciona.

Los permisos son necesarios, en particular, cuando existen múltiples tipos de usuarios conectándose a la base de datos.
