Taller: Interactuando con PSQL

En este taller podrás comenzar a interactuar con PostgreSQL a través de la línea de comandos o terminal.

El comando que vas a usar es psql. En Windows, ejecuta el archivo psql.exe

Realiza los siguientes pasos:

    Entra a tu Postgres con el comando:

    psql -u postgres -W -h localhost

    Coloca tu clave
    Lista las bases de datos con \l
    Crea tu base de datos con:

    CREATE DATABASE prueba;

    Cámbiate de base de datos con \c prueba;
    Crea una tabla con:

    CREATE TABLE usuarios (id SERIAL, nombre VARCHAR, PRIMARY KEY(id));

    Inserta varios registros
    Lista los registros
    Abandona el terminal con \q


Puntos de verificación

Para confirmar que tu experiencia ha sido llevada a cabo con éxito, revisa los siguientes puntos:

    Verifica la tabla: puedes usar la consulta \d+ usuarios para ver la tabla usuarios
    Consulta registros: verifica tus registros con un SELECT * FROM usuariosy luego presiona q para salir
